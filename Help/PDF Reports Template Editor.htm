AxiomCoders PDF Reports Template Editor is tool used to create template files for your reports. This tool is designed so anyone can use it with simplicity in mind.
This is stand-alone GUI application written in .NET and presents WYSIWYG editor. Template files created can be used by any kind of AxiomCoder PDF Reports package used for generating (e.g. you can use these files on our PDF Factory designed for Linux).

Creating templates for your reports is somehow different that what you used to see in other reporting tools so be sure to check video tutorials and read help to understand its full potential.

Advantages of PDF Reports Template Editor

- Editor is created to be very easily understood. This is WYSIWYG editor meaning that report that you create will look same when generated (populated) with appropriate data.
- There is Preview option which will show how your report will look like while creating it.
- Supports gradients and shapes. (that are natively supported by PDF format)
- Support for background images
- There are no section and subsections on report page giving you freedom to design your document like drawing a image.
- Advanced tools for organizing your report items on page (e.g. aligning, sizing, placing).
