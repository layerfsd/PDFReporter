Precalculated item is designed to calculate result from given data by formula you entered. So if you have data like product prices and you want to calculate average value or to calculate discounts in percents or any other mathematic formula combined with some of our rules.

To create Precalculated item, select it from object tool bar and place it on page inside desire container (Static or Dynamic balloon) like shown in picture below.

To set specific properties for Precalculated balloon, select it in page like shown in picture below.

Specific properties for precalculated item are:

ForeColor, is color of the text.

Font, is font name and size use to draw text.

Text, is displayed when Formula doesn't exists.

Formula, mathematic formula combined with our specific rules. Consult example below.

Formula example:

Formula is combination of predefined functions for getting data and calculate it, and standard mathematic symbols.

Predefined functions are:

AVG(DataStream.Column), count average value from given DataStream and Column .

SUM(DataStream.Column), sums values from given DataStream and Column.

MIN(DataStream.Column), represents minimum value from given DataStream and Column .

MAX(DataStrem.Column), represents maximum value from given DataStream and Column.

DECTOBIN(DataStream.Column), will sum all data from DataStream and Column, and result will be in binary format.

DECTOHEX(DataStream.Column), will sum all data from DataStream and Column, and result will be in hex format.

Standard mathematic symbols are:

( ) + - * / 0123456789

Some examples of finished formulas:

SUM(Product.Price) - sum all prices

( SUM(Product.Price) + 100 ) * 0.2 - sum all prices adds 100 and counts 20% off result.

 

Important: Do not place text that do not apply this rules, that formula will be bad and wont return anything as a result.
