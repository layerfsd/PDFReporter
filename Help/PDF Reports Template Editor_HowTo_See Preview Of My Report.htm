To see how will your project look like once when it is generated, you can do the following:

This is our project that we want to see in the preview mode:

1. We need to set number of items editor should create in preview, and for that we need General Options: "Edit->Preferences->General"

2. Now set desired number like shown on the picture in our case it is 6 items.

4. Now press preview button on the toolbar like shown on the picture, and you will see preview of your project.

NOTE!

Remember, in preview mode function are limited, to continue working with your project please exit preview mode by pressing preview button again.
