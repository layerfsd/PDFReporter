Minimum Hardware Requirements:
AxiomCoders LANProtector has modest hardware requirements and should run on virtually any hardware which is able to run supported Operating Systems

Recommended Hardware Requirements:
1GHz x86 or x64 processor
512MB of system memory
1GB of free hard drive space
one or more network adapters

Supported Operating Systems (x86 and x64 versions):*
Microsoft Windows 2000 Professional (with at least Service Pack 4 installed)
Microsoft Windows 2000 Server (with at least Service Pack 4 installed)
Microsoft Windows XP (with at least Service Pack 1 installed)
Microsoft Windows Server 2003
Microsoft Windows Home Server
Microsoft Windows Vista
Microsoft Windows Server 2008

Other Required Software:
Microsoft .Net Framework 2.0 – not included in the installation package and can be separately downloaded and installed from:
Microsoft .Net Framework 2.0 for x86 Operating Systems
Microsoft .Net Framework 2.0 for x64 Operating Systems

CACE Technologies WinPcap 4.0.2 – included in the installation package

 

*Microsoft Windows 9x Operating Systems aren’t supported; Microsoft Windows 7 and Microsoft Windows Server 2008 R2 may work but won’t be explicitly supported until officially released
