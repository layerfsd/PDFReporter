Date Time is object that will display time or date in format you desire. That time will be when you placed object on page.

To create Date Time select it from object tool bar and place it on page like shown in the picture below.

To set specific properties for Date Time object, select it on the page like shown in the picture below.

Specific properties for Date Time are:

ForeColor, color of the Text.

Font, font name and size.

FormatValue, is formula that contains symbols which will be converted to the actual data.

Formula symbols are:

yyyy - Long year format

yy - Short year format

dddd - Long date format

dd - Short date format

HH - Hour symbol

mm - Minute symbol

ss - Seconds symbol

You can also add some off you custom text to the formula to make it more interesting or understandable.
