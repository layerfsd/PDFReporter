Dynamic image is object that represents image from source that is given in time of generating pdf. For example, image is given from data base. Dynamic image must be pleased in side of container (Static or Dynamic balloon). To create image, select it from object toolbar like shown on picture below.

To set specific parameters of Dynamic image, select it like shown in the picture below.

Specific options of the Dynamic image item are:

SourceColumn, represents column where data is.

SourceDataStream, represent Data stream where data is.

ImageType, here you select type of image source. You can have images in files and SourceColumn represents file name then, or if SourceColumn is image bytes then select this to be Data Type
