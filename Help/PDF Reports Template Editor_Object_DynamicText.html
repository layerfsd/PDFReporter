Dynamic text is used to display data from another source, data base for example. It's defined by DataStream and Column.

To create Dynamic text, select it from object tool bar like shown in picture below. Then place it on page in desired location inside of container (Static or Dynamic balloon)

To change specific properties for Dynamic text, select it on page like shown in picture below.

Specific properties for Dynamic text are:

ForeColor, color of the text.

Font, name and size of the font used to display text.

SourceDataStream, DataStream name used to identify extern text.

SourceColumn, Column name used to identify extern text.
