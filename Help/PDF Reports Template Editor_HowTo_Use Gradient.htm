To use gradient for Rectangle Shape, you need to do following:

1. Place rectangle shape on desired location, click on it to be selected

2. Set property "Use Gradinet" to be true, like shown on the next picture. Rectangle will become black and white gradient, because those are default colors.

3. To change colors that are used for gradient, select property "Gradient Definition", like shown on the next picture, when you press button next to it, it will open property window.

4. In the property window you can chouse what color gradient should use, and start, end and direction of the gradient. When finished press OK button.

This is how it will look like after we set all of the desired parameters.
