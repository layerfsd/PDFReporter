Rectangle shape is object that have pure visual effect. It can be one color, or gradient.

To create Rectangle shape, select it from objects toolbar, and place it on page inside container (Static or Dynamic balloon) by clicking desire location and drag desire size of shape. Like shown in picture below.

To set specific properties of Rectangle shape, select it on page like shown in picture below.

Specific properties for Rectangle shape are:

FillColor, is color of the filled area of the shape, and it's shown only if gradient isn't set.

GradientDefinition, opens window where you can chouse custom gradient colors and direction.

StrokeColor, is color for drawing stroke of the shape.

StrokeWidth, is width of the line representing stroke of the object.

UseGradient, tells shape to use or not use gradient. (When used, fill color is ignored)

UseStroke, tells shape to use or not use stroke.
