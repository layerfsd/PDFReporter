AxiomCoders PDF Factory is core library used for generating PDF files.

There is no installers for this library as this library is one .dll file (.so on Linux).
Both Windows and Linux users need to just download this library and copy it to place where it should be used or put it into PATH.

This library has unlimited trial period but will put "This is DEMO" watermark on each page. When you purchase PdfFactory you will get serial number to be used for removing "This is DEMO" watermark.

This library is written in native C language and can be used from any other programming language.
