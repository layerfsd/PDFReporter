Dynamic balloon represents container that is dynamic in way that when is filled with data it can grow and can create himself to fill page as long as he has data to display or if you set its filling capacity. His growing capability is restricted with placed Static balloons, they can't overlap.

Important: you can in one parent balloon place only ONE dynamic balloon, otherwise you must please another static balloon and than inside it another dynamic balloon. (For further help on how balloons appears and rules on how they appear in final PDF, consult StepByStep part of this help)

To create Dynamic balloon, select it from objects toolbar and place it on page by clicking and dragging desire size. Like shown in picture below.

To set specific properties select Dynamic balloon like in picture below.

Specific properties for Dynamic balloon are:

CanGrow, this restrict balloon size if set to false. If set to true balloon will grow no mater how much data is given; if set to false instead it will generate another like it balloon and continue to fill data inside of new balloon.

FillCapacity, Set number of balloons dynamically generated inside this balloon.

BorderProps, set desire border on the balloon.

SourceDataStream, name of the Data stream where data is.
