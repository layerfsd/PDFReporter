Page number is object that showing current page number in format you defined. There is only one rule for page number use this symbol "{P}" without quotes. For example: if FormatValue is something like this: "Page -{P}-" then in generated PDF file the actual text is: "Page -1-" for the first page.

To create Page number, select it from object toolbar like shown in picture below, and place it inside desired container (Static or Dynamic balloon)

To set specific properties for the Page number item, select it on the page like shown in picture below.

Specific properties of the Page number are:

ForeColor, is color used to draw text.

Font, is font used to draw text.

FormatValue, is formatting use to place actual page number in you custom formatted text. (Symbol "{P}" is replaces with current page number)
