Default properties for all object are:

Name, is name of object so you can easily organize your project .

WidthInUnits, is width of object .

HeightInUnits, is height of object .

LocationInUnitsX, is location on X axis inside objects parent .

LocationInUnitsY, is location on Y axis inside objects parent .

Rotation angle, is object rotation angle and it is represented in degrease .

Measuring unit, is objects measuring unit in which objects dimensions and locations are calculated .

Dock, is objects place of docking if you want it to be docked.
