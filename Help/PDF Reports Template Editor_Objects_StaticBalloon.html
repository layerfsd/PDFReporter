Static balloon is a container object that will hold other items placed inside him.

To draw static balloon select it from the object toolbar as shown in picture above, then click on page and drag it on desired size.

To set options or change size of the balloon select it as shown in picture below. As you can see properties on the property window to the right are changed to meet balloon specification.

Specific properties or Static balloon are:

FillCapacity, determine maximum number of child balloons (not items) inserted in him. Zero mean unlimited

AvailableOnEveryPage, sets if this balloon should appear on every page or just on first.

BorderProps, sets if the border should be visible, what border should be visible, colors and size of the borders.

SourceData Stream defines Data stream name which signals to PDF generator that in this balloon exists item that will need some data from that Data Stream.
