Objects toolbar have two parts, non object and object part.

Non object part consists:

Arrow item, it's used to get back from selected object to arrow.

Zoom control, used to set predefined zooms. 50%, 100%, 150%, 200%. For precision zoom use mouse wheel.

Preview mode, used to get program in preview mode which shows you how your report would look like after is generated to PDF file.

Objects part consists:

Static balloon, static container used to draw Header, Footer, Margins, Head lines...

Dynamic balloon, used to draw items that dynamically grow repeat it's self by getting and displaying data

Static text, is simple text that displays as it is drawn in editor.

Dynamic text, uses extern data from predefined DataStream and Column

Static image, uses image from hard drive in specific folder.

Dynamic image, uses image that is externally given from corresponding DataStream and Column

Counter, dynamically count it's repeating during generating process.

Precalculated item, uses formula to display it's self.

Date-Time, when placed uses Date, Time in standard windows time format, or if you want you can set formula to display time or date in any way you like following simple rules.

Page number, counts pages that are generated.

Rectangle shape, it's simple rectangle that you can set color, stroke or even gradient.

Once object is added you can set it's default or specific properties for that particular object.
