Counter is used for example to place number of each row in table. It can be set to have start value, end value, interval.

To create Counter, select it from object toolbar, and place it on the page inside container (Static or Dynamic balloon) like shown in the picture below.

To set specific properties for the Counter, select it on the page like shown in the picture below.

Specific properties for the Counter are:

ForeColor, color of the text.

Font, name and size of the font use to display the text.

StartValue, start value of the counter.

EndValue, end value of the counter. (He won't count over this number)

Looping, loop determines if you want to count from start if you get to the end value or just stop counting.

IntervalValue, interval is value of the step used to count.

FormatValue, format for displaying final text. use next symbols ({S} - Start value, {E} - End value, {C} - actual counting)

Example:

"Something {C} of {E} with begin at {S}" gives next result,

"Something 10 of 100 with begin at 0" for example. (Start = 0, End = 100, Counter value = 10)

Of course you can put only {C} symbol and get pure counting.
