AxiomCoders PDF Reports is complete solution for generating advanced reports in PDF file format.
Main goal with PDF Reports is to provide you with necessary tools to generate complex and good looking reports which you can use as your promotional catalogs, price lists, etc. This way PDF Reports is trying to move job of creating reports from programmers to designers.

AxiomCoders PDF Reports consists of several individual packages and this help contains information on all of them. Some packages are sold separately and it can be possible that you don’t have all of them installed.

As AxiomCoders PDF Reports give you more freedom in creating templates that will be used for generating PDF files please be sure to read our help file or check video tutorials as creating templates in our tools is different than you probably used to with other reporting solutions.

 

Who should use PDF Reports?

 

1)       Developers that want to give their applications ability to generate PDF files from any kind of data.

2)       Developers that found some limitations in other Reporting software

3)       Developers that want to give their reports “life” with good design

 

This tool is primary made for developers. Most of packages included in this solution are made for developers and they purpose is to be integrated into some other application.

 

Simple usage scenario would be like this:

1)       Designer uses PDF Reports Editor to create templates for PDF files

2)       Developer uses some PDF Reports package (e.g. PDF Reports .NET Component) and integrates it into his application. After that his application can generate advanced PDF files from templates with any kind of given data with any number of pages.

 

Who should not use PDF Reports?

 

1)       If you are looking for tool that can convert various formats into PDF file format this tool is not for you

2)       If you are trying to design one pdf page and then print it this tool is not for you.

3)       This tool is designed to be integrated into other applications. If you are looking for stand-alone tool that can generate PDF files by manually feeding data by using PDF template files please contact us as this tool is under development at the moment.

 

We highly recommend you to check samples on what kind of reports you can get with PDF Reports online as they can give you clearer picture on usage of this solution.
