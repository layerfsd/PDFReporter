Minimum Hardware Requirements:

AxiomCoders PDF Factory has lowest possible hardware requirements and should run on virtually any hardware which is able to run supported Operating Systems.

Recommended Hardware Requirements:

None specific. PDF Factory is simple native library that should run fine on any supported Operating System if downloaded exactly for that operating system.

Supported Operating Systems (x64 and x86 versions):

Microsoft Windows 2000 Professional
Microsoft Windows 2000 Server
Microsoft Windows XP
Microsoft Windows Server 2003
Microsoft Windows Home Server
Microsoft Windows Vista
Microsoft Windows Server 2008

PDF Factory should work on Microsoft Windows 95 and Microsoft Windows 98 but we have not performed any tests on that operating systems.

PDF Factory should work on Linux operating systems but as this requires special builds please contact us.

Other Required Software

None
