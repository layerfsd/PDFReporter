Static text is object that is used to draw regular static line of text, like head lines, names of the column in table or some comments. In word everything that you place in design time and want to be there exactly how you put it.

To draw Static text, select it from object toolbar and place it on page inside of Container (Static or Dynamic balloon). Like shown in picture below.

To set specific options for Static text, select it like shown in picture below.

Specific properties for Static text are:

Text, is text that be shown.

Font, name and size of the font you want to use. (Do not use Unicode font, they are NOT supported at this moment)

ForeColor, is color in which text would be drawn.
